Automatisation 360° avec GitHub Actions

3ème événement TADx le 3 décembre à 19h30 à MAME (Tours)

Posted by F3Y & S6E on November 11, 2019 · 1 min read

GitHub vous connaissez ? Mais si c'est cette petite boite qui gère nos sources ;).

Eh bien ce n'est pas que ça, venez découvrir comment GitHub se lance dans l'aventure de l'intégration et déploiement continue avec GitHub Actions.

Nabil Bouabdallah (solution engineer chez GitHub) nous fera découvrir tout cela au travers de cas d'usages concrets !

La présentation sera suivie d’une discussion libre autour de ces thèmes.

Les inscriptions se passent sur eventbrite.

Cela se déroule dans les locaux de MAME à partir de 19h30 le mardi 3 décembre.
